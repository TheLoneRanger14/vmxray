How to build reglookup library
-------------------------------

* Requirements
- libiconv 
- sconstruct python module

* Setup environment
- Update PATH to include python binary path and python scripts path.
- Python scripts path is generally <python_install>\Scripts


* Run command 'scons'
- This will build reglookupLib.dll (reglookupLib.so on linux)


Using the DLL
---------------
- Setup PATH to include libiconv2.dll and reglookupLib.dll
